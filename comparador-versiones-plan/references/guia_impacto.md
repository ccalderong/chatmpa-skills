# Guía de Clasificación de Impacto

## Criterios detallados y casos difíciles

Esta guía complementa el vocabulario controlado del skill. Úsala cuando la clasificación no sea obvia.

---

## `Fortalece` — Verde

El cambio hace el plan más específico, más accionable o mejor respaldado que la versión previa.

**Casos claros:**
- Se añade una meta numérica donde antes no había ninguna (ej. "se monitoreará" → "se medirá cobertura de coral en ≥ 6 sitios cada primavera").
- Se nombra una institución responsable donde antes era genérico (ej. "las autoridades" → "CONANP en coordinación con la institución gestora").
- Se incorpora un protocolo o índice específico donde antes había una intención vaga.
- Se actualiza una cifra con datos más recientes que reflejan mejor la realidad del AMP.
- Se añade una sección nueva con contenido sustantivo que el plan anterior no cubría.

**Caso de borde — Añadido que no Fortalece:**
Si se añade una sección nueva pero su contenido diluye o contradice compromisos existentes (ej. se añade una zona de uso permitido que superpone con una zona núcleo), clasificar como `Debilita`.

---

## `Neutraliza` — Azul

El cambio no altera los compromisos, metas ni protocolos del plan. Es cosmético, administrativo o de estilo.

**Casos claros:**
- Corrección ortográfica o gramatical sin cambio de significado.
- Sustitución de nombre completo por abreviatura (ej. "Parque Nacional Cabo Pulmo" → "PN Cabo Pulmo").
- Reorganización del orden de párrafos sin cambio de contenido.
- Actualización de encabezados o numeración de secciones.
- Cambio de tiempo verbal sin cambio de compromiso (ej. "se realizará" → "se realiza").

**Caso de borde — texto reutilizado verbatim:**
Si un párrafo de diagnóstico se traslada sin cambios de una versión de hace 10+ años, clasificar como `Neutraliza` (el cambio en sí es nulo), pero en la celda `Comparación` señalar explícitamente que el contenido no fue actualizado pese al tiempo transcurrido — eso es información valiosa para el revisor humano aunque el impacto del *cambio* sea neutro.

---

## `Debilita` — Rojo

El cambio reduce la efectividad, especificidad o ambición del plan respecto a la versión previa.

**Casos claros:**
- Se elimina una meta numérica y se reemplaza por una formulación vaga (ej. "recuperar ≥ 18% de cobertura para 2030" → "promover la recuperación del coral").
- Se elimina la mención de una institución responsable específica.
- Se elimina un protocolo o procedimiento concreto.
- Se suaviza un compromiso (ej. "prohibido" → "restringido", "se implementará" → "se evaluará la posibilidad de").
- Desaparece una sección completa con contenido sustantivo.
- Se introduce una inconsistencia (cifra contradictoria, zona que desaparece sin explicación).

**Caso de borde — eliminación de contenido redundante:**
Si lo eliminado era duplicado exacto de otra sección, clasificar como `Neutraliza` en lugar de `Debilita`.

**Caso de borde — simplificación que mejora:**
Si se elimina texto confuso o contradictorio y se reemplaza por algo más claro y específico, clasificar como `Fortalece` aunque haya contenido eliminado.

---

## Tabla de decisión rápida

| Situación | Impacto |
|---|---|
| Nueva meta numérica o indicador específico | `Fortalece` |
| Meta numérica eliminada o vuelto vago | `Debilita` |
| Corrección ortográfica / abreviatura / reorden | `Neutraliza` |
| Texto de diagnóstico reutilizado sin actualizar | `Neutraliza` (señalar en Comparación) |
| Institución responsable nombrada por primera vez | `Fortalece` |
| Institución responsable eliminada | `Debilita` |
| Protocolo específico añadido | `Fortalece` |
| Protocolo específico eliminado | `Debilita` |
| Cifra actualizada con datos más recientes | `Fortalece` |
| Cifra contradictoria sin explicación | `Debilita` (→ `Inconsistencia`) |
| Zona o subzona que desaparece sin reemplazo | `Debilita` (→ `Inconsistencia`) |
| Sección nueva con contenido sustantivo | `Fortalece` |
| Sección eliminada con contenido sustantivo | `Debilita` |
